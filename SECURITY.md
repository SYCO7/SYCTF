# Security Policy

## Supported Versions

SYCTF is in early public release. The latest tagged release is supported for security fixes.

## Reporting a Vulnerability

If you discover a security issue in SYCTF itself, please open a private security report through GitHub Security Advisories.

If private reporting is not available, open an issue with minimal reproduction details and avoid publishing exploit details until maintainers respond.

## Educational Use Disclaimer

SYCTF is intended for:

- CTF competitions
- personal labs
- explicitly authorized security testing

Do not use SYCTF against systems you do not own or do not have written permission to test.

Users are responsible for complying with all applicable laws, platform terms, and competition rules.
